# Gameplay & Design Guidelines

## Core Loop
1. **Gather**: Collect ingredients from the world (farming, fishing, mining).
2. **Cook**: Process ingredients in the kitchen using recipes.
3. **Serve**: Provide dishes to guests to earn Gold and Experience.
4. **Progress**: Use Gold to upgrade tools, buy decorations, and unlock new areas.

## Tone & Art Direction
- **Vibe**: Cozy, cooperative, whimsical.
- **Characters**: Zundamon and Zundapal are the main guides. Dialogue should be friendly and helpful.
- **Visuals**: Vibrant colors, "kawaii" aesthetic, consistent emoji usage in UI (e.g., 🍙, 🍡, ✨).

## UI Principles
- **Clarity**: Use icons and emojis to represent items and actions.
- **Feedback**: Provide visual (FX) and textual (Toasts/Popups) feedback for all player actions.
- **Consistency**: All UI logic should ideally go through `StarterPlayerScripts` and use shared configs.
