# 🎀 Zundamon's Kitchen - Electra Setup Guide

## 🚀 Quick Start (3 Steps Only!)

### 1. Install 🛠️
```powershell
cargo install rojo --locked
```
*(or download from https://github.com/rojo-rbx/rojo/releases)*

### 2. Open Game 🎮
- Open `beantown.rbxl` in Roblox Studio
- Plugins → Rojo → Sync → `default.project.json`

### 3. Play & Edit ✨
- Click Play Solo
- Edit scripts in `src/` folder
- Changes sync automatically!
