# Zundamon's kItchen

The Rojo project lives in **`Zundamons-kItchen-GitHub-Build/`** (nested inside this repo).

## Windows — first time on a new PC?

**Start here:** [`Zundamons-kItchen-GitHub-Build/docs/windows-first-time-setup.md`](Zundamons-kItchen-GitHub-Build/docs/windows-first-time-setup.md)

Quick clone path after setup:

```
C:\Users\<you>\Zundamons-kItchen\Zundamons-kItchen-GitHub-Build
```

## Sync code into Roblox Studio

```powershell
cd $HOME\Zundamons-kItchen\Zundamons-kItchen-GitHub-Build
git pull origin main
npm run rojo:serve
```

In Studio: open experience **108617605497926** → **Plugins → Rojo → Connect** → Play → look for `[ROJO SYNC OK]` in Output.

Full troubleshooting: [`Zundamons-kItchen-GitHub-Build/docs/rojo-sync-troubleshooting.md`](Zundamons-kItchen-GitHub-Build/docs/rojo-sync-troubleshooting.md)
